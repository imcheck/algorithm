# Problem 1

N \* N 정수 배열이 주어질 때 대각선의 합을 구하는 코드를 작성하세요.

입력 예)

```
4
1 4 2 3
6 2 3 1
1 5 2 3
1 1 1 1
```

출력 예)

```
18
```
