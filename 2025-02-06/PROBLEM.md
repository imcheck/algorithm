# Problem 1

N \* M 배열이 주어질 때 상하 반전해서 출력하세요.

입력 예)

```
2 3
1 2 3
4 5 6
```

출력 예)

```
4 5 6
1 2 3
```

# Problem 2

N \* M 배열이 주어질 때 시계 방향으로 180도를 돌려서 출력하세요.

입력 예)

```
2 3
1 2 3
4 5 6
```

출력 예)

```
6 5 4
3 2 1
```

# Problem 3

N x M, M x N 배열 2개가 주어질 때 행렬 곱셈의 결과를 출력하시오.

입력 예)

```
2 3
1 2 3
4 5 6
1 2
3 4
5 6
```

출력 예)

```
22 28
49 64
```
